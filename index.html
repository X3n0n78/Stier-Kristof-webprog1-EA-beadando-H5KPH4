Projekt bemutatása

Ez a honlap a Web-programozás-1 előadás beadandó feladatának keretében készült. A menüben található oldalak mindegyike egy-egy különböző webes technológiát mutat be (CRUD táblázat, HTML5 API-k, ChartJS, AJAX, OOJS, React).

  • Táblázat: CRUD műveletek, rendezés, szűrés
  • HTML5: Web Storage, Web Workers, Geolocation, stb.
  • ChartJS: Diagram ábrázolása
  • AJAX: API-val kommunikáló alkalmazás
  • OOJS: Objektumorientált JavaScript példa
  • React: Két egyszerű SPA alkalmazás
